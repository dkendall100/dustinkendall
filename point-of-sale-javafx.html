Java Point of Sale

  • Dustin Kendall

  • April 30th, 2017

Description

A point of sale system I created for a school project. I used object oriented design concepts to accomplish this. This application allows employees to create an order, remove items, tender cash, and prints a receipt to a text file when completed. Managers are able to void previously completed orders.

  • JavaFX
